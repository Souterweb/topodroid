MULTI-SHOT EDIT

This dialog is opened by the "light bulb" button from the Surevy Data window in multi-shot mode.

It has the following actions

  • renumbering the shots stations. The first selected shot must be a leg
  • swapping stations
  • coloring. Only if all selected shots are splays
  • mark the splays as either regular (-), x-section (X), horizontal (H) or vertical (V). Only available if all selected shots are splays
  • hide the shots in either the plan, or the profile view, or both. Only if all selected shots are splays
  • computing the plane that fits the shots
